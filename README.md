## Beschreibung

Diese Anwendung ist eine Umsetzung des beliebten Videospiels Tetris.

Der Spieler muss fallende Blöcke geschickt platzieren, um vollständige Reihen zu bilden und diese vom Spielfeld zu entfernen. Mit zunehmender Spielzeit steigt die Geschwindigkeit der fallenden Blöcke, wodurch das Spiel immer anspruchsvoller wird.

Die Anwendung wurde mit JavaFX entwickelt und bietet eine benutzerfreundliche Oberfläche.

----------

## Spielprinzip

Der Spieler steuert verschiedene Tetrominos (Tetris-Blöcke), die von oben auf das Spielfeld fallen.

Mögliche Aktionen:

-   Nach **links(A)** oder **rechts(D)** bewegen
    
-   **Rotieren(W)**
    
-   **Schneller fallen lassen(S)**
    
Sobald eine Reihe vollständig gefüllt ist, wird sie entfernt und der Spieler erhält Punkte.

Das Spiel endet, wenn neue Blöcke aufgrund fehlenden Platzes nicht mehr auf dem Spielfeld erscheinen können.

## Spiel starten

Das Spiel wird über die Klasse **Launcher** gestartet.

1.  Projekt in der IDE öffnen.
2.  Die Klasse **Launcher.java** auswählen.
3.  Die Anwendung über die Run-Funktion der IDE starten.
4.  Das Spielfenster öffnet sich automatisch.
